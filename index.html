Presentación KPI - Power BI

Descripción General

Los reportes desarrollados en Power BI representan la capa de presentación del análisis del Indicador Mensual de Ticket Promedio de Ventas Totales por Canales Digitales (Página Web y Aplicación Móvil) para Falabella Perú.

Para fines académicos, la información utilizada corresponde a un caso ficticio construido a partir de datos simulados generados mediante herramientas de inteligencia artificial, con el objetivo de replicar un escenario realista de análisis de ventas digitales.

Los reportes fueron diseñados con un enfoque ejecutivo, priorizando una interfaz intuitiva que facilite la exploración de la información y el análisis del indicador.

Principales características

  • Storytelling con datos: cada reporte sigue una secuencia lógica que facilita la comprensión del comportamiento del indicador.
  • Análisis descriptivo: se incluyen indicadores como ticket promedio mensual, monto total de ventas y número de transacciones.
  • Filtros dinámicos: los usuarios pueden explorar la información aplicando filtros por fecha y canal digital.

Modelo de datos

El modelo de datos fue construido en Power BI a partir de las tablas publicadas desde Microsoft Fabric, utilizando una estructura en estrella que optimiza el análisis de información.
